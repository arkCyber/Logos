# 用户指南

## 概述

LOGOS是一个功能强大的电子表格和文档编辑器，支持Typst输出、模板管理、协作编辑等功能。

## 快速开始

### 安装

1. 克隆仓库:
```bash
git clone https://github.com/your-org/LOGOS.git
cd LOGOS
```

2. 安装依赖:
```bash
npm install
```

3. 启动开发服务器:
```bash
npm run dev
```

4. 启动后端服务:
```bash
cd spreadsheet-service
cargo run
```

### 首次使用

1. 打开应用后，点击"新建文档"创建新文档
2. 选择模板或从空白开始
3. 开始编辑内容

## 功能介绍

### 电子表格编辑

#### 基本操作

- **创建工作表**: 点击"新建工作表"按钮
- **编辑单元格**: 双击单元格进行编辑
- **公式输入**: 在单元格中输入`=SUM(A1:A10)`
- **格式设置**: 右键单元格选择格式选项

#### 高级功能

- **条件格式**: 选择单元格范围，设置条件格式规则
- **图表创建**: 选择数据范围，插入图表
- **数据透视表**: 选择数据范围，创建数据透视表
- **Excel导入/导出**: 支持导入和导出Excel文件

### 文档编辑

#### Typst编辑器

1. 打开Typst预览编辑器
2. 在左侧编辑器中输入Typst代码
3. 右侧实时显示PDF预览
4. 点击"编译"按钮手动编译

#### 导出选项

1. 点击"导出"按钮
2. 选择导出格式（PDF、SVG、PNG、HTML等）
3. 设置导出质量和页面范围
4. 点击"导出"按钮下载文件

### 模板管理

#### 使用模板

1. 打开模板库
2. 浏览或搜索模板
3. 点击"使用"按钮应用模板
4. 设置模板变量（如有）

#### 创建模板

1. 创建文档并编辑内容
2. 定义模板变量
3. 保存为模板
4. 添加模板元数据

#### 模板评分和评论

1. 在模板详情页点击评分
2. 添加评论和反馈
3. 查看其他用户的评论

### 双栏编辑器

#### 使用双栏编辑器

1. 打开双栏编辑器
2. 左侧是Tiptap富文本编辑器
3. 右侧是Typst PDF预览
4. 编辑内容自动同步到预览

#### 工具栏功能

- **文本格式**: 粗体、斜体、下划线、删除线
- **标题**: H1、H2、H3
- **列表**: 无序列表、有序列表
- **表格**: 插入表格、添加行列、删除行列
- **对齐**: 左对齐、居中、右对齐

#### 同步功能

- **状态同步**: 工具栏按钮根据光标位置自动亮灭
- **数据同步**: 编辑内容自动编译为PDF（防抖500ms）
- **视觉同步**: 点击编辑器内容自动滚动到PDF对应位置

## 协作功能

### 实时协作

1. 点击"共享"按钮
2. 邀请协作者
3. 实时查看其他用户的编辑
4. 查看光标位置和编辑历史

### 评论和修订

- **添加评论**: 选中内容，点击"添加评论"
- **修订模式**: 启用修订模式查看修改历史
- **接受/拒绝**: 接受或拒绝修订建议

## 快捷键

### 通用快捷键

- `Ctrl+S`: 保存文档
- `Ctrl+Z`: 撤销
- `Ctrl+Y`: 重做
- `Ctrl+C`: 复制
- `Ctrl+V`: 粘贴
- `Ctrl+X`: 剪切

### 编辑器快捷键

- `Ctrl+B`: 粗体
- `Ctrl+I`: 斜体
- `Ctrl+U`: 下划线
- `Ctrl+1`: 标题1
- `Ctrl+2`: 标题2
- `Ctrl+3`: 标题3

## 常见问题

### Q: 如何导入Excel文件？

A: 点击"文件"菜单，选择"导入Excel"，选择文件即可。

### Q: 如何导出为PDF？

A: 点击"导出"按钮，选择PDF格式，设置导出选项，点击"导出"。

### Q: Typst编译失败怎么办？

A: 检查Typst代码语法，查看错误信息，修正后重新编译。

### Q: 如何创建自定义模板？

A: 编辑文档后，点击"保存为模板"，填写模板信息即可。

### Q: 协作编辑如何使用？

A: 点击"共享"按钮，输入协作者邮箱，发送邀请即可。

## 故障排除

### 应用无法启动

1. 检查Node.js版本（需要>=16）
2. 删除node_modules重新安装
3. 检查端口是否被占用

### 后端服务无法连接

1. 检查Rust是否安装
2. 检查数据库文件权限
3. 查看后端日志

### Typst编译错误

1. 检查Typst语法
2. 查看错误详情
3. 确保Typst包已安装
