### Luckysheet 导入问题修复

#### 问题分析
- **错误**: `Failed to resolve entry for package "luckysheet"`
- **原因**: luckysheet 包的 package.json 配置在测试环境中无法正确解析
- **影响**: 2 个测试文件无法运行

#### 修复方案
在测试文件中添加 Spreadsheet 组件的 mock：

```typescript
// Mock Spreadsheet component to avoid luckysheet import issues
vi.mock('../Spreadsheet.vue', () => ({
  default: {
    name: 'Spreadsheet',
    template: '<div class="mock-spreadsheet">Mock Spreadsheet</div>'
  }
}));
```

#### 修复结果
- ✅ Editor.typst.test.ts 现在可以正常运行（31 个测试通过）
- ✅ Editor.integration.test.ts 保持跳过状态（需要完整 DOM 环境）
- ✅ 所有其他测试文件正常运行
- ✅ 生产环境不受影响
