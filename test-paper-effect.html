Word 风格编辑器

这是一个模拟 Microsoft Word 编辑界面的示例。你现在看到的是：

  • 灰色画布背景 - 使用 #a5a5a5 颜色，与 Word 完全一致
  • 白色 A4 纸张 - 794×1123 像素，标准 A4 比例
  • 纸张阴影 - 多层阴影营造立体感
  • 居中布局 - 纸张在画布上水平居中
  • 合适的边距 - 上下 96px，左右 120px（Word 默认值）

排版特性

文字使用 Calibri 字体，12pt 大小，1.6 行高，提供舒适的阅读体验。

标题使用不同的字号：

  • H1: 22pt 粗体
  • H2: 16pt 粗体
  • H3: 13pt 粗体

交互体验

你可以直接在这个白色区域输入文字，就像在 Word 中一样。滚动页面时，纸张会保持居中。

在窄屏幕上（宽度小于 900px），纸张会自动适应屏幕宽度，确保在移动设备上也有良好的体验。

技术实现

这个效果通过以下 CSS 实现：

  • editor-content-wrapper: 灰色画布容器
  • document-canvas: Flexbox 居中布局
  • editor-mount: A4 白纸，带阴影和边距

完全符合 Word 2019/2021 的视觉设计规范。
