## 功能对比分析

### 原有功能
- Typst 编译器集成（基础）
- Typst 渲染器（PNG/PDF）
- 字体加载器
- 语法高亮
- 模板引擎
- 基础参考文献管理

### 新增功能（按优先级）

#### 高优先级功能（已完成 ✅）

1. **Counter 系统** (`counter.rs`)
   - 自动编号和计数功能
   - 支持多种编号样式（阿拉伯数字、罗马数字、字母、符号）
   - 作用域支持
   - 更新历史记录
   - Typst 语法解析

2. **Grid 布局系统** (`layout.rs`)
   - 网格布局引擎
   - 自适应列宽计算
   - 支持分数单位（fr）
   - 行列配置
   - HTML 和 Typst 格式输出

3. **Stack 布局系统** (`layout.rs`)
   - 水平和垂直堆叠布局
   - 间距控制
   - 填充配置
   - 响应式渲染

4. **数据加载功能** (`data_loader.rs`)
   - JSON 格式支持
   - YAML 格式支持
   - CSV 格式支持
   - TOML 格式支持
   - CBOR 格式支持
   - XML 格式支持
   - 数据路径查询
   - 缓存机制

5. **HTML 导出格式** (`export.rs`)
   - 完整 HTML 文档生成
   - CSS 样式支持
   - 响应式设计
   - 打印优化

6. **SVG 导出格式** (`export.rs`)
   - 矢量图形导出
   - 单页和多页支持
   - 可配置 DPI
   - 字体嵌入选项

7. **Query 查询系统** (`query.rs`)
   - 元素类型查询
   - 属性过滤
   - 位置过滤
   - Typst 查询语法解析
   - 查询结果缓存

8. **State 状态管理系统** (`state.rs`)
   - 键值对状态存储
   - 作用域支持
   - 只读保护
   - 持久化支持
   - 历史记录和撤销
   - Typst 状态语法解析

#### 中优先级功能（已完成 ✅）

1. **可视化形状** (`shapes.rs`)
   - Circle（圆形）
   - Rectangle（矩形）
   - Ellipse（椭圆）
   - Square（正方形）
   - 描边和填充支持
   - 旋转变换
   - SVG、HTML、Typst 格式输出

2. **高级数学功能** (`advanced_math.rs`)
   - Binomial（二项式系数）
   - Primes（质数生成和检测）
   - Cancel（取消符号）
   - Cases（情况语句）
   - Matrix（矩阵）
   - LaTeX、Typst、HTML 格式输出

3. **符号和表情符号支持** (`symbols.rs`)
   - 数学符号（∞, ∂, ∇, ∑, ∏, ∫ 等）
   - 希腊字母（α, β, γ, δ, ε, θ, λ, μ, σ, φ, ω 等）
   - 箭头符号（→, ←, ↑, ↓ 等）
   - 运算符（∪, ∩, ⊂, ⊃, ∈, ∉ 等）
   - 货币符号（$, €, £, ¥ 等）
   - 表情符号（😊, 😂, ❤️, 👍 等）
   - 符号搜索和替换
   - LaTeX 到 Typst 转换

4. **小型大写字母和智能引号** (`text_formatting.rs`)
   - 小型大写字母转换
   - HTML 格式支持
   - 智能引号（英语、法语、德语、瑞典语）
   - 撇号处理
   - 文本转换（大写、小写、标题大小写、句子大小写）
   - 引号样式检测

#### 低优先级功能（已完成 ✅）

1. **Typst 插件系统** (`plugin.rs`)
   - 插件元数据管理
   - 沙箱执行环境
   - 权限验证系统
   - 插件缓存机制
   - 批量加载支持

2. **渐变和平铺支持** (`gradient.rs`)
   - 线性渐变（Linear）
   - 径向渐变（Radial）
   - 锥形渐变（Conic）
   - 渐变停止点（Gradient Stops）
   - 平铺模式（Repeat, Reflect, Clamp）
   - 变换支持（平移、缩放、旋转）
   - CSS、SVG、Typst 格式输出

3. **高级路径操作** (`path.rs`)
   - 路径命令（MoveTo, LineTo, CubicBezier, QuadraticBezier, ArcTo）
   - 路径组合（Union）
   - 路径变换（Offset, Scale, Rotate）
   - 边界框计算
   - 路径长度计算
   - SVG、Canvas、Typst 格式输出
